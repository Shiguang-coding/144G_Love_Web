# 部署说明

1. 绑定号域名
2. 将文件上传到服务器wwwroot
3. 子目录下的所有文件、文件夹赋予读写权限
4. 放行端口及安全组

尽量不用https，可能会出现调用出错的情况




生成后的数据保存在 `data`目录下，以json格式进行存储

预览页面地址：http://106.15.61.102:8001/web.php?id= + json文件名称
